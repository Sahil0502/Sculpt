# Sculpt

Hackathon-ready 1-on-1 performance catalyst with online meetings, offline voice capture, AI insights, and role-based dashboards.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env.local` from `.env.example` and fill in your keys.

3. Start the dev server:

```bash
npm run dev
```

## Scripts

- `npm run dev` - start local dev server
- `npm run build` - production build
- `npm run start` - start production server
- `npm run lint` - lint project

## Notes

- API routes return mocked data when required keys are missing.
- Supabase and OpenAI wiring are ready for real credentials.
